# WP Stats

A collection of WordPress statistics

## Download file sizes

This data shows the size of the WordPress zip file, with and without installed themes, and the size difference compared to the previous version. Only versions 4.0 and later are listed.

![](/images/wp-download-file-sizes.png)

## Days between releases

This data shows point releases since version 1.0 and the number of days since the previous release.

![](/images/wp-days-since-release.png)


## Plugins in the review queue

This data shows a history of how many plugins are waiting for review. Where available, the estimated wait is also recorded.

![](/images/wp-review-queue.png)
